# Système de Gestion de Tickets

## Installation et Lancement Rapide (Windows)
1. Assurez-vous que **Docker Desktop** est lancé.
2. Double-cliquez sur le fichier `run_project.bat`.
3. Attendez la fin du processus (environ 1-2 minutes le temps du build).

## Accès aux interfaces
- **Frontend :** [http://localhost:5173](http://localhost:5173)
- **Administration Django :** [http://localhost:8000/admin](http://localhost:8000/admin)
